================================================================================
                        üîß TROUBLESHOOTING
================================================================================

PROBLEM: Microphone not working
SOLUTION: 
  1. Check browser permissions (click lock icon in address bar)
  2. Make sure microphone is plugged in
  3. Try refreshing the page
  4. Use Chrome or Edge browser

PROBLEM: JARVIS not responding
SOLUTION:
  1. Check internet connection
  2. Speak more clearly
  3. Try rephrasing your command
  4. Check the command list in JARVIS-README.md

PROBLEM: Voice recognition not accurate
SOLUTION:
  1. Reduce background noise
  2. Speak at normal pace
  3. Use a better quality microphone
  4. Move closer to microphone
